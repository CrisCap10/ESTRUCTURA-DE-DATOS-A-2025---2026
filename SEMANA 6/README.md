## SEMANA 6 – LISTAS ENLAZADAS

### Ejercicio 8
Programa en C# que utiliza listas enlazadas para almacenar una cantidad N
de números reales, calcular el promedio y separar los datos en dos listas:
- Menores o iguales al promedio
- Mayores al promedio

### Ejercicio 9
Programa en C# que maneja dos listas enlazadas de enteros y verifica:
- Si tienen el mismo tamaño
- Si tienen el mismo contenido y orden

El programa muestra el resultado según la comparación realizada.
