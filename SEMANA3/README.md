## Semana 3 – Registro de Estudiante (C#)

Programa en consola que permite registrar los datos de un estudiante utilizando
conceptos básicos de Estructura de Datos.

### Funcionalidad
- Ingreso del ID del estudiante
- Ingreso de nombres y apellidos
- Registro de dirección
- Almacenamiento de tres números de teléfono usando un array
- Visualización de los datos ingresados en consola

### Materia
Estructura de Datos
